## üìä Face Recognition Algorithm

### **BlazeFace Model**
- Lightweight ML model optimized for mobile/web
- 6-point facial landmark detection
- Real-time processing capability

### **Comparison Methods**
1. **Procrustes Analysis**: Optimal alignment with rotation compensation
2. **Descriptor-Based**: Geometric feature extraction (distances, angles, centroids)
3. **Hausdorff Distance**: Robust to outliers

### **Weighted Combination**
- Euclidean: 20%
- Descriptor: 60%
- Hausdorff: 20%

### **Requirements**
- Minimum 10 training samples per user
- 70% similarity threshold for authentication
- At least 5 samples must match above threshold
